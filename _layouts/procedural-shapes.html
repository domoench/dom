This site's home page animation is proceduraly generated using javascript, SVG, and CSS keyframe animations.

On each page load the algorithm generates a unique animation and layout of shapes. It randomly subdivides the browser viewport area and, for each sub-area, either paints a shape, or recurses upon it - repeating the process.

The colors and shapes are directly inspired by Inyegumena Nosegbe's design explorations for PUBLIC SERVICE.
